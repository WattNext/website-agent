# Answer Unit Template

```md
title:
category:
trigger:
short_answer:
approved_answer:
expand_if_asked:
cta:
confidence_level:
source_owner:
last_updated:
```

## Notes

- `trigger` can include multiple phrasings or intents.
- `short_answer` should work for fast voice replies.
- `approved_answer` is the default answer body.
- `expand_if_asked` is only for follow-up depth.
- `cta` should suggest the next helpful move.
- `confidence_level` can be `high`, `medium`, or `needs-review`.
